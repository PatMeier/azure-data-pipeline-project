# Azure Data Pipeline Project

## Überblick
Dieses Projekt implementiert eine einfache ETL-Pipeline mit Python und Pandas.

## Technologien
- Python
- Pandas
- Azure Data Lake Storage (optional)

## Pipeline
1. Daten laden (CSV)
2. Transformation (Join + Berechnung)
3. Aggregation
4. Speicherung

## Ausführung
python src/etl_pipeline.py
